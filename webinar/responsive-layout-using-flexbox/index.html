Frontend Starter Pack: Episode 2 – Responsive Layout using Flexbox

I. Details of the event/activity

The webinar titled “Frontend Starter Pack: Episode 2 – Responsive Layout using Flexbox” conducted by Team pixelPlay on December 1, 2v022, will feature Mr. Lawrenz Diestro as the resource speaker. The webinar will provide a comprehensive discussion on how to create responsive layouts using the Flexbox module of CSS. Attendees will learn how to use Flexbox to create a responsive grid system that can adapt to different screen sizes, and how to use Flexbox to create a responsive navigation menu. Mr. Diestro will also discuss the different properties and values of the Flexbox module, and will demonstrate how to use them for responsive layouts.
